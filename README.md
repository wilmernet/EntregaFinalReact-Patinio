
# Tienda de Mascotas

Corresponde a un ejercicio académico en el que entrené REACT, acompañado del equipo de formación de coderhouse

## Instalación

Descarga el proyecto e Instala las dependencias con npm

```bash
  npm install
```
o
```bash
  npm i
```
